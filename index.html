TradeFlow Privacy Policy

Effective Date: [18/02/2026]

1. Data Storage

TradeFlow stores all user data locally on the user's device using Chrome's storage.local API. This includes checklists, trade settings, preferences, and configuration data. No user data is transmitted to any external server.

2. External API Usage

TradeFlow makes optional requests to https://api.frankfurter.app, a free and open-source foreign exchange rate API, to retrieve live currency price data. These requests do not include or transmit any personal user information.

3. No Tracking or Analytics

TradeFlow does not collect analytics, telemetry, tracking data, or personal information. The extension does not use cookies or third-party tracking services.

4. Contact

If you have any questions regarding this privacy policy, please contact the developer through the Chrome Web Store listing.
