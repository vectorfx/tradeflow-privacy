TradeFlow Privacy Policy

Effective Date: 18 February 2026

1. Data Storage

TradeFlow stores user data locally on the user's device using Chrome's storage.local API. This includes checklists, trade settings, preferences, and configuration data.

2. Billing & Subscription

TradeFlow contacts tradeflow-biller.vercel.app to verify subscription status (Pro, Brand Mode) and to open account management pages. These requests use session cookies set when the user logs in on the billing site. No personal data beyond session authentication is transmitted, and no data is used for tracking, analytics, or advertising.

3. External API Usage

TradeFlow makes optional requests to https://api.frankfurter.app, a free and open-source foreign exchange rate API, to retrieve live currency price data. These requests do not include or transmit any personal user information.

4. Third-Party Fonts

The extension loads the Geist font from Google Fonts (fonts.googleapis.com). This may result in a network request to Google's servers. No personal information is sent by the extension as part of this request. See Google's Privacy Policy for details on how Google handles font requests.

5. No Tracking or Analytics

TradeFlow does not collect analytics, telemetry, tracking data, or personal information. The extension does not use cookies or third-party tracking services.

6. Contact

If you have any questions regarding this privacy policy, please contact the developer through the Chrome Web Store listing.
